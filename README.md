# Simulateur Fiscal Freelance

Comparateur fiscal pour freelances en France : Micro-entreprise, EI, EURL et SASU, avec options IR/IS et montage holding.

**Production** : [simulateur-freelance-ten.vercel.app](https://simulateur-freelance-ten.vercel.app)

## Stack

- React 18 (CDN) + Babel standalone
- Pas de bundler, un seul fichier JSX
- Déployé sur Vercel (auto-deploy depuis GitHub)

## Statuts simulés

- Micro-entreprise
- EI (Entreprise Individuelle) — option IR ou IS
- EURL — option IR ou IS
- SASU — option tout salaire ou salaire + dividendes
- SASU + Holding
