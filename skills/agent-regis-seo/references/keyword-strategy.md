# Keyword Strategy

## What Success Looks Like

The owner has a clear keyword map: which pages target which queries, where the gaps are, and what content to create next. Every page has a primary keyword, secondary keywords, and search intent alignment. The content architecture reflects the complete semantic field of "simulateur de revenus" across all statuts français.

## Your Approach

**Semantic Field Mapping** — Map the complete universe of queries around income simulation in France: "simulateur freelance", "simulateur portage salarial", "calcul salaire net cadre", "comparaison SASU EURL", "simulation impôt micro-entreprise", "combien je gagne en freelance", etc. Group by search intent (informational, transactional, comparative).

**Page-Keyword Assignment** — Each existing page gets a primary keyword target and 3-5 secondary keywords. Identify pages that compete with each other (cannibalization) and pages that should exist but don't (content gaps).

**Search Intent Alignment** — Ensure each page matches its target intent: a simulator page should answer "combien" questions, a comparison page should answer "quel statut choisir", an explanation page should answer "comment ça marche".

**Competitor SERP Analysis** — Identify who ranks for target queries, what content format they use, what structured data they have, and where the opportunity gaps are.

**Content Roadmap** — Prioritized list of pages/content to create, based on: search volume, competition level, conversion potential, and topical authority gaps.

## Memory Integration

Maintain the keyword map in memory — which pages target what, current estimated positions, content gaps identified. Track progress over time.

## After the Session

Log keyword decisions, new targets identified, content gaps found, and any cannibalization issues detected.
