# Funnel Analysis

## What This Achieves

Marc analyse les parcours utilisateur en se basant sur les données de tracking disponibles. Identifie les points de friction, les abandons, et propose des améliorations pour optimiser la conversion.

## What Success Looks Like

- Mapping clair des funnels principaux
- Identification des points de drop-off avec données
- Hypothèses sur les causes de friction
- Recommandations d'amélioration priorisées par impact
